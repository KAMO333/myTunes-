## 📖 About This Project

**myTunes** is a high-performance, responsive landing page for a fictional music streaming service.  
The project was refactored from legacy **jQuery** into **modern Vanilla JavaScript** using **Object-Oriented Programming (OOP)** principles to improve maintainability, scalability, and performance.

It demonstrates clean frontend architecture, native browser APIs, and automated UI testing practices.

---

## 🚀 Industrial Upgrades

- **⚙️ jQuery Removal**  
  Eliminated external dependencies to reduce bundle size and improve load performance.

- **🧱 OOP Architecture**  
  UI behavior encapsulated inside a `NavigationController` class for structured state management.

- **🌊 Native Smooth Scrolling**  
  Replaced jQuery animations with the native `window.scrollTo` API.

- **🧪 Automated Testing**  
  Jest + JSDOM test suite validates UI interactions and navigation logic.

- **📁 Professional Structure**  
  Organized into scalable `src/` and `tests/` directories.
